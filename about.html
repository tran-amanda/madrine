Getting Started with jQuery

By Amanda Tran · December 2020

Discover how to write basic jQuery statements and utilize jQuery’s distinguished selector feature to perform basic tasks. Some examples outlined include traversing the Document Object Model and accessing data of HTML elements.

This article covers the basic fundamentals of jQuery API (syntax, usage, etc.). To see jQuery in action, refer to my demo Simple Responsive Menu with jQuery.

Writing Basic jQuery Statements

All jQuery statements start with the dollar symbol ($), followed by code that selects HTML elements called selectors, wrapped in quotation marks and parenthesis. Following the dollar symbol ($) is the selector, then a period to denote a method and its respective parenthesis. Finally, the jQuery statement ends with a semicolon. 

  
      $("selector").method();

In most cases, a jQuery statement which has a selector but does not include a method will run a DOM search for HTML elements, returning the matching elements. Keep in mind that usage of the dollar symbol ($) is not limited only to selecting HTML elements. This is because the dollar symbol ($) is shorthand for jQuery() which can also manipulate JavaScript objects, albeit with limitations.

jQuery Selectors

Selectors in jQuery are written in the same syntax as CSS Selectors, unlike plain JavaScript. Thus, selecting an HTML element with their class (.), id (#), or tag name follows the same syntax rules. 

   
      $("#example"); //select elements with the id named example

jQuery Methods

Methods in jQuery follow the selector with a period symbol (.) and have different operations based on the method being called. 

   
      //insert Your Content at the end of every div element
      $("div").append("Your Content");

The Benefits of jQuery

The main benefit of using jQuery over plain JavaScript is in its many features which allow users to code for efficiency and speed.

For example, selecting a specific set of HTML elements nested within many other elements is made simple by traversing methods. If you needed to traverse up the DOM Tree, the parent() method selects the element one level higher than its selector element. Combining this traversal method with other operations allows users to make powerful changes to multiple HTML elements with shorter code.

Additionally, selecting and accessing data is also simpler with jQuery. An example demonstrating this technique is with the css() method, which I use in my jQuery demo. In the demo, I select an HTML element and check the values of its style properties. See this example on how to use jQuery methods for the web in my demo, Simple Responsive Menu with jQuery.
