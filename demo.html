Simple Responsive Menu with jQuery

By Amanda Tran · December 2020

With CSS media queries and the ability to alter an HTML’s element styling through the css() and removeAttr() methods, you can create a responsive and interactive navigation menu without removing or adding new elements to the DOM. Using jQuery allows you to easily select and access an existing HTML element to edit an attribute by manipulating its properties and values.

The css() method also accesses the values of DOM elements’ style properties, which is used to check if the menu is in an open or closed state for this demo.

Demo on JSFiddle

1.

Set up the JavaScript

To make the menu icon clickable and enable the jQuery library for JavaScript, add two <script> tags to the <head> tag in your HTML document. The first <script> tag enables the jQuery library by using the CDN link listed on the official website. For this example, I use the slim and minified version of the latest stable release (jQuery Core 3.5.1). 

    
      <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js" integrity="sha256-4+XzXVhsDmqanXGHaHvgh1gMQKX40OUvDEBTu8JcmNs=" crossorigin="anonymous"></script>

Add the second <script> tag, which contents will be replaced the function to change the menu appearance later in the demo. 

    
      <script>
          //code to add...
      </script>

2.

Create the Navigation Menu

The menu consists of an unordered list of items which will be styled with CSS to look like a proper navigation bar. The style attribute of the <ul> tag and its children (the <li> tags) will be altered with jQuery to display an opened menu state later. 

    
      <ul id="nav">
        <li>Link 1</li>
        <li>Link 2</li>
        <li>Link 3</li>
        <li id="icon" onclick="changeMenu()">
          &#10006;
        </li>
      </ul>

Note: Notice the <li> tag for our menu icon includes the onclick event attribute to call the changeMenu() function, which sets new style attribute property-values later on. The onclick event attribute attaches the function call to its proper HTML element (the menu icon) for click events.

3.

Style the Navigation Menu

The unordered list is styled like a navigation bar using a flexbox layout and by removing the list decoration on links with the list-style property. Notice that the HTML element for the menu icon is hidden by default as the display property is set to none. However, with the media query, menu links are hidden instead and only the menu icon is visible at the 600px breakpoint. 

    
      #nav { /*styles for nav bar*/
        display: flex;
        justify-content: space-around;
        align-items: center;
        flex-direction: row;
        flex-wrap: nowrap;
        width: 100%;
        list-style: none;
      }
    
      li#icon { /*hide menu button default*/
        display: none;
      }
    
      @media (max-width: 600px) { /*hide links and show burger menu for smaller screens*/
        #nav li {
          display: none;
        }
    
        li#icon {
          display: block;
        }
      }

Note: For this demo, we use CSS statements only to style the default closed appearance of the navigation menu. The styles to display an open state are added with jQuery, and the script cannot make any direct changes to your CSS file or embedded CSS. Keep in mind how styles will be overwritten or reverted with jQuery’s ability (and limitations) in regards to accessing DOM elements.

4.

Add the Menu’s Open State

Adjust the navigation menu to display as fullscreen with collapsed links by changing the #nav element (remember, the unordered list) style attribute CSS properties using jQuery’s css() method. Set column and 100vh values for the flex-flow and height properties to achieve the open appearance. To unhide the links, set the display property to block for #nav li elements. This change is applied to every HTML element matching the jQuery selector. 

    
      //set new style attribute properties and values for open appearance
      $("#nav").css({ 
        "flex-flow": "column", //collapse links 
        height: "100vh" //nav container to fullscreen
      });
      $("#nav li").css("display", "block" ); //show links

Note: Notice how the css() method creates display changes by manipulating the style attribute within the selected element’s HTML tag. To see this in action, watch the HTML document change in your browser’s Inspector View as the script runs.

css() Method Syntax

The css() method syntax has two parameters: css(property, value), however the value parameter is optional. The css() method will not set new values without a value argument, and instead returns the value of the first matched property. The property parameter is required.

To set multiple properties and their values using the css() method, the statement must be enclosed within braces {} inside the method parenthesis. The property name and value are also separated with colons (:) and commas (,) are instead used to separate each property-value statement.

5.

Add the Menu’s Closed State

Instead of using the css() method to style the navigation menu back to its default appearance, use the removeAttr() method to remove the style attribute changes. By removing the style attribute from the selected elements, the HTML elements only use the CSS styles set from the stylesheet or styles in the <head> tag. 

    
      //remove style changes to revert menu to default closed appearance
      $("#nav").removeAttr("style");
      $("#nav li").removeAttr("style");

style attribute properties and their values from the selected HTML elements. Be sure to style the default navigation menu appearance with CSS within your stylesheet or embedded CSS, or you will need to write extra code with the css() method to account for styles within the style attribute.

6.

Check the Menu State with If/Else Statements

To ensure that the menu responds with the appropriate changes, we use an if/else statement which checks the value of the flex-direction property for the #nav element. The value of the flex-direction property is accessed every time the function is called, or in other words— when the menu icon is clicked. Since the flex-direction property is set to row by default from the stylesheet, the if statement only executes when the menu is open; as the flex-direction property value is set to column in the open state. 

    
      if ($("#nav").css("flex-direction") === "column") { //menu is in open state
        //statements...                 
      } else { //menu is in closed state
        //statements...
      }

7.

The Final Code Output

The HTML code includes embedded CSS styles in the <head> tag, along with embedded JavaScript which includes the changeMenu() function that updates the respective HTML elements’ style attribute to display open and closed appearances. The jQuery library is enabled in the first <script> tag. 

    
      <!DOCTYPE html>
      <html lang="en">
      
        <head>
          <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js" 
          integrity="sha256-4+XzXVhsDmqanXGHaHvgh1gMQKX40OUvDEBTu8JcmNs=" 
          crossorigin="anonymous"></script>
          <script>
            function changeMenu() { //function to run when user clicks on menu icon
      
              if ($("#nav").css("flex-direction") === "column") { //menu is in open state
                //remove jquery css changes to revert menu to closed state
                $("#nav").removeAttr("style");
                $("#nav li").removeAttr("style");
      
              } else { //menu is in closed state
                $("#nav").css({
                  "flex-direction": "column", //arrange column/list links
                  height: "100vh" //nav container to full screen
                });
                $("#nav li").css("display", "block"); //show links 
              }
            }
      
          </script>
          <style>
            #nav {
              /*styles for nav bar*/
              display: flex;
              justify-content: space-around;
              align-items: center;
              flex-direction: row;
              flex-wrap: nowrap;
              width: 100%;
              list-style: none;
            }
      
            li#icon {
              /*hide menu button default*/
              display: none;
            }
      
            @media (max-width: 600px) {
      
              /*hide links and show burger menu for smaller screens*/
              #nav li {
                display: none;
              }
      
              li#icon {
                display: block;
              }
            }
      
          </style>
        </head>
      
        <body>
          <ul id="nav">
            <li>Link 1</li>
            <li>Link 2</li>
            <li>Link 3</li>
            <li id="icon" onclick="changeMenu()">
              &#10006;
            </li>
          </ul>
        </body>
      
      </html>
